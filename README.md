# Rust 学习笔记

个人 Rust 学习笔记，按《Programming Rust》章节组织，用大白话讲清楚 Rust 的核心概念。

```

05_错误处理/

07_结构体/
08_枚举与模式/
09_特型与泛型/
10_运算符重载/
11_实用工具特型/
12_闭包/
13_迭代器/

17_并发/
18_异步编程/
19_宏/

21_外部函数/
```

## 笔记特点

- **口语化表达**：像跟朋友聊天一样写，不端着
- **代码驱动**：每个概念都有可运行的示例
- **理解原理**：不只讲「怎么用」，更讲「为什么」
- **对比学习**：相似概念放一起，帮你理清区别

## 如何使用

1. 按章节顺序阅读，或直接跳到感兴趣的主题
2. 每个章节的 `概述.md` 是入口，提供整体视角
3. 具体概念文件深入讲解单个知识点
4. 代码示例可以直接复制到 Rust Playground 运行
